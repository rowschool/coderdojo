Curriculum

What to Expect

Each student at CoderDojo is different and works at their own pace. Software development and creative fields cover such a broad range of topics - ranging from animal sprites all the way to the digital logic of zeroes and ones. The curriculum guide below is just that - a guide.

Although each programming concept below is designed to take a day, it might not take you all day. Everyone moves at their own pace, so do you! We'll be there to support you along the way.

Scratch

If this is your first time programming, we suggest you get your start in a drag-and-drop programming tool called Scratch. You can get started with a tutorial. Just press play!

Start with Scratch!

If this is your first day at CoderDojo, but you've done some programming before and don't know where to start, raise your hand. A mentor can guide you through your own thought process. Alternatively, you can build something according to today's theme in Scratch.

Start Programming

If you really want to start programming, you're going to need to start writing code instead of dragging blocks around. Scratch teaches the concepts of programming - conditionals, loops, and blocks - and the following sites below implement those concepts in text.

Code Combat

Crawl, fight, and puzzle your way through Kithgard Dungeon, the first level of Code Combat! You can learn either Python or JavaScript and hack, dart, slash, and cast your way to victory!

Play Code Combat!

Codecademy

Codecademy is a free platform for learning dozens of languages. You'll need to create an account to get started.

Get an IDE

As you get more comfortable writing code, you're going to need an Integrated Development Environment (IDE). An IDE is like a text editor, except it highlights syntax, makes it easier to integrate your code into a project, and provides small hints and suggestions within your development environment. Codeanywhere is an editor just like the ones the pros use - and you can use it anywhere! You'll need to create an account to get started.
